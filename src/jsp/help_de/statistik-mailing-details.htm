Online-Hilfe: Statistik - Mailing-Statistik - Details

Das Register zeigt statistische Informationen zum ausgewählten Mailing an. Im oberen Bereich stellen Sie ein, welche Empfänger Sie erfassen wollen, unten erscheinen die Details dazu. Über das Export-Symbol () speichern Sie die angezeigten Daten als CSV-Datei auf Ihrer Festplatte. Diese Datei können Sie zum Beispiel mit Excel weiterverarbeiten. Die Auswahllisten und Optionsfelder haben folgende Bedeutung:

  • Zielgruppe: Mit der Zielgruppe wählen Sie aus, für welchen Teil der Empfänger Sie eine Statistik sehen wollen. Dabei spielt es keine Rolle, ob Sie eine bestimmte Zielgruppe beim Versand der E-Mails verwendet haben. Diese Einstellung wirkt sozusagen als Filter. Damit eine Änderung wirksam wird, klicken Sie auf den Button OK.
  • nur Netto-Klicks Standardmäßig summiert das System alle Klicks auf einen messbaren Link auf, auch wenn ein Empfänger mehrmals den Link anklickt. Mit der Option nur Netto-Klicks zählt das System dagegen pro Empfänger nur einen Klick, auch wenn dieser mehrmals klickt. Damit eine Änderung wirksam wird, klicken Sie auf den Button OK.

Die tabellarische Auswertung zeigt die Mausklicks auf die messbaren Links und Daten zum Versand der E-Mail an:

  • URL: Der erste Abschnitt listet sämtliche messbaren Links im Mailing auf. In der ersten Spalte sehen Sie die entsprechende Web-Adresse, in der zweiten Spalte Anzahl registrierter Klicks. Mit dem Pfeilsymbol () vor der Web-Adresse öffnen Sie die Zielseite in einem neuen Browserfenster.
  • Gesamt (Empfänger/Klicks): Der zweite Abschnitt zeigt das Verhältnis von Empfängern zu Klicks. Meistens wird es mehr Empfänger als Klicks geben, denn nicht jeder ruft die verknüpfte Webseite auf.
  • Geöffnete E-Mails: Die Anzahl der geöffneten Mails ist nur aussagekräftig, wenn das Mailing ausschließlich im HTML-Format versendet wurde. Das Öffnen von Text-Mails durch die Empfänger kann OpenEMM aus technischen Gründen nicht registrieren.
  • Abmeldungen: Der Abschnitt weist aus, wieviele Empfänger den Link zum Abmelden in der E-Mail verwendet haben.
  • Bounces: Der Wert von Bounces gibt an, wie viele E-Mails eine falsche Empfängeradresse hatten und deshalb an den OpenEMM zurückgeschickt wurden. Es kann durchaus etliche Stunden dauern, bis eine solche negative Rückmeldung nach dem Versand eintrifft.
  • Empfänger: Die Anzahl der Empfänger, an die das Mailing insgesamt verschickt wurde.

Mit dem Link Admin- und Test-Empfänger-Klicks entfernen bereinigen Sie die Statistik, so dass nur Rückmeldungen von "echten Empfängern" zu sehen sind.
