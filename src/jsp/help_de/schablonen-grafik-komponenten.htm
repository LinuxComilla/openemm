Online-Hilfe: Schablonen - Grafik-Komponenten

Dieser Eingabedialog ist identisch mit dem Dialog unter der Rubrik Mailings/Grafik-Komponenten. Sie übertragen damit Grafiken zum Server, die Sie dann in Mailings mit HTML-Format verwenden können. Die Eingabefelder und Buttons haben folgende Bedeutung:

  • Neue Komponente: In das Eingabefeld tragen Sie den Dateinamen der Grafik auf Ihrer Festplatte ein. Der Button Durchsuchen ruft den Dateiauswahldialog von Windows auf, der ausgewählte Dateiname erscheint dann im Eingabefeld. Um die Übertragung zum Server zu starten, klicken Sie auf den Button Hinzufügen.
  • Übersichtliste: In der Liste sind sämtliche für dieses Mailing bereits übertragenen Grafiken aufgelistet. Links sind bei jedem Eintrag der Dateiname und der Typ der Datei (Grafikformat) angegeben. Rechts sehen Sie eine Vorschau der Grafik. Über den Button Löschen entfernen Sie eine Grafik-Komponente aus dem Mailing.
    Achtung: Das System löscht die Komponente ohne weitere Rückfrage.
