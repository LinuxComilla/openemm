Online-Hilfe: Mailings - Messbare Links
In der Übersicht sind sämtliche Links (HTML-Verweise) aufgeführt, die im Text des Mailings, also in einem Textbaustein im Register Inhalt, enthalten sind. Die einzelnen Spalten der Tabelle haben folgende Bedeutung:
  • URL: Die Spalte zeigt die Web-Adresse (URL = Uniform Resource Locator), die Sie im Mailing eingetragen haben. Dabei muss es sich um eine gültige Adresse auf einem beliebigen Web-Server handeln. Um den Link zu überprüfen, klicken Sie auf das Pfeilsymbol () vor dem Eintrag. Der Browser öffnet ein neues Fenster und zeigt darin die Zielseite des URL an.
  • Beschreibung: Die Beschreibung erscheint nur innerhalb des E-Marketing Managers und soll Ihnen die Zuordnung des messbaren Links erleichtern.
  • Messbar: Diese Spalte zeigt an, in welchen E-Mails des Mailings E-Marketing Manager den Link als messbar formatiert. Die vier erlaubten Werte haben die folgende Bedeutung:
    • nicht messbar: Der Link erscheint unverändert im Mailing. Der Empfänger der E-Mail kann den Link zwar anklicken und die entsprechende Webseite in seinem Browser aufrufen. Der E-Marketing Manager kann diese Aktion jedoch nicht auswerten.
    • nur Text-Version: Der Link wird nur in der Textversion der E-Mails als messbar formatiert. Die Empfänger von HTML-Mails (HTML und Offline-HTML) sehen den Original-Link wie bei der Einstellung nicht messbar.
    • nur HTML-Version: Der Link wird nur in der HTML-Version (HTML und Offline-HTML) der E-Mails als messbar formatiert. Die Empfänger von Text-Mails sehen den Original-Link wie bei der Einstellung nicht messbar.
    • Text- und HTML-Version: Der Link wird in allen E-Mails (Text, HTML und Offline-HTML) als messbar formatiert. Das ist die Voreinstellung, die meistens sinnvoll ist.
  • Aktion: Die Spalte zeigt, wie der E-Marketing Manager reagiert, wenn ein Anwender den messbaren Link anklickt. Eine Aktion muss vorher unter der Rubrik Aktionen definiert werden. Dort können Sie auch prüfen, aus welchen Einzelanweisungen die Aktion besteht.
Klicken Sie eine URL, eine Beschreibung oder das Symbol Bearbeiten () am Ende eines Eintrags an, so öffnet der E-Marketing Manager den Eingabedialog mit den Details zu dem messbaren Link. Die Eingabefelder und Auswahllisten haben dieselbe Bedeutung wie in der Übersicht. Wenn Sie einen Wert ändern, klicken Sie abschließend auf den Button Speichern. Wollen Sie dagegen zurück zur Übersicht der messbaren Links, klicken Sie auf Abbrechen.
