Online-Hilfe: Mailings - Datei-Anhänge

Eine E-Mail kann beliebig viele Dateianhänge haben, zum Beispiel eine Preisliste im PDF-Format. Im oberen Bereich des Eingabedialogs legen Sie neue Dateianhänge an, unten sehen Sie eine Liste der vorhandenen Dateianhänge. Wenn Sie mit den Eingaben für einen neuen Anhang fertig sind, klicken Sie auf den Button Hinzufügen. Die Eingabefelder und Auswahllisten haben folgende Bedeutung:

  • Datei-Anhang: In dieses Eingabefeld tragen Sie den Dateinamen des Anhangs auf Ihrer Festplatte ein. Mit dem Button Durchsuchen rufen Sie den Dateiauswahldialog von Windows auf. Name und Pfad der ausgewählten Datei werden dann automatisch in das Eingabefeld eingetragen.
  • Name des Datei-Anhangs: Das Eingabefeld dient dazu, dem Dateianhang einen Namen zu geben, unter dem er an das Mailing angehängt wird. Standardmäig trägt das System hier den Dateinamen aus dem Eingabefeld darüber ein. Es macht Sinn, hier einen prägnanten und kurzen Dateinamen zu wählen. Achten Sie aber auch die korrekte Endung, etwa .pdf für ein PDF-Dokument. Sonst kann der Empfänger den Dateianhang womöglich nicht öffnen oder anzeigen.

Die Liste der Datei-Anhänge zeigt zu jedem Eintrag eine Reihe von Details und einige Bedienelemente an. Wenn Sie Änderungen vornehmen, dann klicken Sie abschließend auf Speichern. Wollen Sie einen Anhang entfernen, so klicken Sie auf den Button Löschen. Achtung: Der Löschvorgang startet ohne Rückfrage.

  • Datei-Anhang: Rechts neben dem Dateinamen des Anhangs sehen Sie ein Download-Symbol (). Falls Sie einen Dateianhang auf Ihrer Festplatte nicht mehr haben, klicken Sie dieses Symbol an, um die Datei vom Server herunterzuladen.
  • MIME-Typ: Der MIME-Typ ist eine internationale Kennzeichnung für das Format der angehängten Datei. Der MIME-Typ eines PDF-Dokuments ist beispielsweise application/pdf.
  • Originalgröße: Die Größe der Datei auf der Festplatte in KByte.
  • Größe in E-Mail: Als Anhang zu einer E-Mail muss eine Datei auf besondere Weise formatiert werden. Dadurch vergrößert sich eine Datei etwa um den Faktor 1,5. Dieses Feld sagt Ihnen, wie groß der Dateianhang exakt ist. Für die Übertragungszeit beim Empfangen der E-Mail ist dieser Wert entscheidend. Erst das E-Mail-Programm des Empfängers wandelt beim Speichern den Anhang wieder in die Originaldatei zurück.
