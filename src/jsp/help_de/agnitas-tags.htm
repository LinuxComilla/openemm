Online-Hilfe: AGNITAS-Tags
Über die AGNITAS-Tags personalisieren Sie Ihre Mailings, fügen Links zum Abmelden ein, oder passen den Inhalt an verschiedene Zielgruppen an. Die AGNITAS-Tags fügen Sie in den Text der E-Mail ein. Im Gegensatz zu HTML-Tags verwenden die AGNITAS-Tags eckige Klammern zur Kennzeichnung. Beim Versenden der einzelnen E-Mails ersetzt der OpenEMM die AGNITAS-Tags individuell durch die entsprechenden Texte, etwa den Vornamen des Empfängers. So erzeugen Sie zum Beispiel Betreffzeilen mit dem Namen des Empfängers, fügen die E-Mail-Adresse des Empfängers ein oder bauen Links zum Abmelden oder zum Aufruf des Empfängerprofils ein. Im einzelnen stehen folgende AGNITAS-Tags zur Verfügung:
  • [agnTITLE type=x]: Setzt eine komplette Anrede in die E-Mail ein. Statt x geben Sie als Parameter 1 oder 2 an:
    • [agnTITLE type=1] ergibt: Sehr geehrte(r) Frau/Herr Nachname
    • [agnTITLE type=2] ergibt: Liebe(r) Vorname Nachname
    Bitte beachten Sie: Natürlich können Sie auch individuelle Anredearten definieren (siehe Handbuch). In diesem Fall geben Sie als Parameter die ID der gewünschten Anredeart an: [agnTITLE type=164] ergibt: Text der Anredeart mit der ID 164
  • [agnFIRSTNAME]: Setzt den Vornamen des Empfängers ein.
  • [agnLASTNAME]: Setzt den Nachnamen des Empfängers ein.
  • [agnEMAIL]: Setzt die E-Mail-Adresse des Empfängers ein.
  • [agnSUBSCRIBERCOUNT]: Setzt die Anzahl der Mailing-Empfänger ein.
  • [agnDATE type=x]: Setzt das aktuelle Datum ein. x steht für das Format:
    • [agnDATE type=0]: Ergibt das Datumsformat 3.4.2004
    • [agnDATE type=1]: Ergibt das Datumsformat 04/03/2004
    • [agnDATE type=2]: Ergibt das Datumsformat Dienstag, 3. April 2004
    • [agnDATE type=3]: Ergibt das Datumsformat 2001-04-03
  • [agnDB column="..."]: Setzt den Inhalt eines beliebigen Feldes aus der Empfängerdatenbank ein. Mit dem Parameter column= geben Sie den Namen des Feldes an. Ein Beispiel (das Feld SCHUHGROESSE muss natürlich definiert sein): [agnDB column="SCHUHGROESSE"]
  • [agnIMAGE name="..."]: Das Bild mit dem angegebenen Namen aus den Grafik-Komponenten des Mailings einbinden. Achtung: Sie geben hier nicht den Dateinamen des Bildes an, sondern den Namen, den Sie beim Einbinden des Bildes im Register Grafik-Komponenten vergeben haben.
  • [agnUNSUBSCRIBE]: Setzt einen personalisierten Link zur Abmeldung von der Mailing-Liste ein. Aus <a href="[agnUNSUBSCRIBE]">Klicken Sie hier zur Abmeldung</a> wird in der E-Mail zum Beispiel: <a href="http://rdir.de/uq?000009560008acfe0008acf68a295">l; Klicken Sie hier zur Abmeldung</a>
  • [agnPROFILE]: Setzt einen personalisierten Link zur Änderung der persönlichen Daten ein. Das funktioniert analog zu [agnUNSUBSCRIBE].
  • [agnONEPIXEL]: Setzt einen personalisierten Link ein, der als URL in einem <img src="..."> ein transparentes 1-Pixel-GIF zurückliefert. Damit kann dann festgestellt werden, wie oft der HTML-Newsletter geöffnet wurde. Ein Beispiel: <img src="[agnONEPIXEL]"> ergibt in der E-Mail <img src="http://rdir.de/g?00000551000005180000071d2d3">
