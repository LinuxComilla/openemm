Online-Hilfe: Kampagnen - Neue Kampagne

In diesem Eingabedialog legen Sie eine neue Kampagne an. Wenn die Eingaben abgeschlossen sind, erzeugen Sie mit dem Button Speichern die neue Kampagne, der Sie dann einzelne Mailings zuordnen können. Die einzelnen Eingabefelder haben folgende Bedeutung:

  • Name: Bestimmt den Namen der Kampagne. Dieser Name erscheint später in der Kampagnen-Übersicht und erlaubt die Auswahl.
  • Beschreibung: Gibt Ihnen die Möglichkeit, Kommentare zur jeweiligen Kampagne einzutragen.
