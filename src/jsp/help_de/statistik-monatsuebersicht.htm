Online-Hilfe: Statistik - Monatsübersicht
Die Monatsübersicht fasst alle Mailings in einem Monat zusammen. Über die beiden Pfeilsymbole ( und ) blättern Sie einen Monat vor oder zurück. Wenn Sie Details zu einem Mailing anzeigen wollen, klicken Sie den Namen in der Spalte Mailing an. Der E-Marketing Manager zeigt dann die Einstellungen zu diesem Mailing an (identisch mit dem Aufruf von Mailing - Details).
