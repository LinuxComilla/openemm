Online-Hilfe: Schablonen - Details

Über die Detailansicht passen Sie eine vorhandene Schablone an. Wenn Sie mit den Änderungen fertig sind, schließen Sie die Aktion mit einem Klick auf den Button Speichern am Ende des Dialogs ab. Um die Schablone zu entfernen, klicken Sie auf den Button Löschen am Ende des Dialogs. Die Eingabefelder und Auswahllisten haben folgende Bedeutung:

  • Name: Bestimmt den Namen der Schablone. Dieser Name erscheint später im OpenEMM immer dort, wo Sie eine Schablone auswählen können.
  • Beschreibung: Zeigt den Kommentar zur Schablone an.
  • Mailing-Liste: Mit diesem Drop-Down-Menü legen Sie fest, welcher Mailingliste Sie diese Schablone zuweisen.
  • Kampagne: Gibt Ihnen die Möglichkeit, die Schablone einer bestehenden Kampagne zuzuordnen.
  • Mailing-Typ: Hier haben Sie die Möglichkeit, die Schablone einem bestimmten Mailing-Typ zuzuordnen. Sie haben die Auswahl zwischen normalen Mailings sowie Datums- und Aktionsgesteuerten Mailings.
  • Zielgruppen: Zeigt zunächst die für diese Schablone eingestellte Zielgruppe an. Standard: Alle Empfänger
  • Hinzufügen: Durch Auswahl im Drop-Down-Menü und Klick auf "Hinzufügen" werden mehrere vordefinierte Zielgruppen in Form einer "oder"-Verknüofung (keine Doppelzustellung bei Überschneidungen) verbunden. Durch Klick auf das Entfernen-Symbol () wird die betreffende Zielgruppe entfernt.
  • Betreff: Wenn Sie ein neues Mailing auf Basis dieser Schablone anlegen, wird dieser Text als Vorschlag eingeblendet. Sie können ihn dann übernehmen, oder durch einen anderen Text ersetzen. Die Wirkung des Betreff sollte nicht unterschätzt werden. Er sollte gut gewählt werden, damit der Empfänger das Mailing öffnet. Beachten Sie, dass hier auch personalisierte Angaben mit den AGNITAS-Tags erlaubt sind.
  • Absender-Adresse: Wenn Sie ein neues Mailing auf Basis dieser Schablone anlegen, wird dieser Text als Vorschlag für die Absender-Adresse eingeblendet. Sie können ihn dann übernehmen, oder durch einen anderen Text ersetzen. Anzuraten ist eine nachvollziehbare E-Mail-Adresse wie z.B. newsdienst@ihreadresse.de. Allgemein erwecken Real-Name-Absender mehr Vertrauen beim Empfänger, also etwa sebastian.mueller@firma.de.
  • Absender-E-Mail: Über dieses Feld bestimmen Sie, welche E-Mail-Adresse als Absender-Adresse standardmäßig im abschließenden Mailing erscheint. Anzuraten ist eine nachvollziehbare E-Mail-Adresse wie z.B. newsdienst@ihreadresse.de. Sie können die E-Mail-Adresse durch einen Namen ersetzen, den die Empfänger dann in ihrem E-Mail-Programm sehen. Im Hintergrund bleibt aber die vollständige Adresse erhalten, so dass Rückantworten auch weiterhin funktionieren. Dazu geben Sie die Adresse im Format "Name" an. Praktisch könnte das zum Beispiel so aussehen: "Hans Hoelzl" . Der Empfänger sieht nur den Namen Hans Hoelzl im Absenderfeld seines E-Mail-Programms. Die Absender-Adresse können Sie später in der Rubrik Mailings in der Detailansicht wieder ändern.
  • Absender-Name: / Antwort-Name: Sie können den Realname der Absender-Adresse und der Reply-To-Adresse (Antwortadresse) unterschiedlich benennen. Die hinter den Realnames stehende E-Mail-Adresse bleibt identisch, es handelt sich lediglich um den angezeigten Namen. Die unterschiedlichen Namen können Sie in den entsprechenden Formularfeldern unter Absender-Name bzw. Antwort-Name eingeben. Wenn Sie nur einen Namen in Absender-Name eingeben, so wird dieser automatisch als Antwort-Name übernommen.
  • Zeichensatz: Die Auswahlliste zeigt die Zeichensätze an, die der OpenEMM zur Verfügung stellt. Für die meisten Fälle dürfte die Voreinstellung ISO 8859-1 die richtige Wahl sein, sie deckt alle westeuropäischen Sprachen ab.
  • Zeilenumbruch nach: Diese Auswahlliste ist nur für Textmails interessant: Der OpenEMM fügt nach der angegebenen Zeichenzahl automatisch einen Umbruch ein.
  • Format: Hier wählen Sie das Mail-Format der Schablone aus: Text-, HTML- oder Offline-HTML. Im Textformat enthält die E-Mail keinerlei Formatanweisungen wie fett oder Schriftgröße und auch keine klickbaren Links. Das HTML-Format erlaubt die bei Webseiten üblichen Formatierungen und auch die Integration von Bildern. Bei externen Dateien (Bildern etc.) enthält die E-Mail nur die Referenz, die Bilddatei selbst ist auf den Server gespeichert. Offline-HTML funktioniert genauso, allerdings sind Bilder und andere externe Dateien Teil der E-Mail (Dateianhänge).
  • Öffnungsrate messen: Um die Öffnungsrate bei HTML-Mailings messen zu können, wird ein "Messpixel" in das E-Mailing aufgenommen. Bei der Öffnung des Newsletters wird dieses Pixel vom Server geladen, was zur Registrierung der Öffnung führt.
  • Text-Version: In diesem Eingabefeld fügen Sie die Text-Version der Schablone ein. Weitere Details dazu lesen Sie im Handbuch. Auch wenn Sie in der vorhergehenden Auswahlliste das HTML oder Offline-HTML als Format gewählt haben, tragen Sie hier einen Text ein. Denn auch die Empfänger von HTML-Mails erhalten zur Sicherheit auch die Textversion.
  • HTML-Version: In diesem Eingabefeld fügen Sie die HTML-Version der Schablone ein. Weitere Details dazu lesen Sie im Handbuch.
  • Speichern": Speichert Ihre Einstellungen bzgl. der überarbeiteten Schablone.
  • Kopieren: Erstellt eine Kopie der jeweiligen Schablone und erleichtert die Erstellung von lediglich leicht modifizierten Schablonen.
  • Löschen: Löscht nach nochmaliger Rückfrage die Schablone.
