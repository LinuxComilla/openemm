Online-Hilfe: Schablonen - Details
Über die Detailansicht passen Sie eine vorhandene Schablone an. Wenn Sie mit den Änderungen fertig sind, schließen Sie die Aktion mit einem Klick auf den Button Speichern am Ende des Dialogs ab. Um die Schablone zu entfernen, klicken Sie auf den Button Löschen am Ende des Dialogs. Die Eingabefelder und Auswahllisten haben folgende Bedeutung:
  • Name: Bestimmt den Namen der Schablone. Dieser Name erscheint später im E-Marketing Manager immer dort, wo Sie eine Schablone auswählen können.
  • Beschreibung: Zeigt den Kommentar zur Schablone an.
  • Mailing-Liste: Mit diesem Drop-Down-Menü legen Sie fest, welcher Mailingliste Sie diese Schablone zuweisen.
  • Kampagne: Gibt Ihnen die Möglichkeit, die Schablone einer bestehenden Kampagne zuzuordnen.
  • Mailing-Typ: Hier haben Sie die Möglichkeit, die Schablone einem bestimmten Mailing-Typ zuzuordnen. Sie haben die Auswahl zwischen normalen Mailings sowie Event- und Regelbasierten Mailings.
  • Zielgruppen: Zeigt zunächst die für diese Schablone eingestellte Zielgruppe an. Standard: Alle Empfänger
  • Hinzufügen: Durch Auswahl im Drop-Down-Menü und Klick auf "Hinzufügen" werden mehrere vordefinierte Zielgruppen in Form einer "oder"-Verknüofung (keine Doppelzustellung bei Überschneidungen) verbunden. Durch Klick auf das Entfernen-Symbol () wird die betreffende Zielgruppe entfernt.
  • Medien-Typ Priorität: Hier sehen Sie, welche Medientypen in welcher Priorität Sie für Ihre Multi-Channel-Kampagne festgelegt haben.

    Zum Ändern der jeweiligen Priorität innerhalb der verschiedenen Medientypen klicken Sie jeweils beim gewünschten Medien-Typ auf die blauen "up-" oder "down"-Pfeile. Nach jedem Klick auf einen Pfeil wird das Browserfenster aktualisiert und zeigt den entsprechenden Medien-Typ um eine Position nach oben (Erhöhung der Priorität) bzw. nach unten (Senkung der Priorität) an.
  • Reiter-E-Mail:
    • Betreff: Tragen Sie einen Text ein, der später beim Versand als Betreffzeile Ihres Mailings in den E-Mail-Programmen der Empfänger erscheinen soll. Diesen Text können Sie später in der Rubrik Mailings in der Detailansicht wieder ändern.
    • Absender-Adresse: Über dieses Feld bestimmen Sie, welche E-Mail-Adresse als Absender-Adresse standardmäßig im abschließenden Mailing erscheint. Anzuraten ist eine nachvollziehbare E-Mail-Adresse wie z.B. newsdienst@ihreadresse.de. Sie können die E-Mail-Adresse durch einen Namen ersetzen, den die Empfänger dann in ihrem E-Mail-Programm sehen. Im Hintergrund bleibt aber die vollständige Adresse erhalten, so dass Rückantworten auch weiterhin funktionieren. Dazu geben Sie die Adresse im Format "Name" an. Praktisch könnte das zum Beispiel so aussehen: "Hans Hoelzl" . Der Empfänger sieht nur den Namen Hans Hoelzl im Absenderfeld seines E-Mail-Programms. Die Absender-Adresse können Sie später in der Rubrik Mailings in der Detailansicht wieder ändern.
    • Zeichensatz: Die Auswahlliste zeigt die Zeichensätze an, die der E-Marketing Manager zur Verfügung stellt. Für die meisten Fälle dürfte die Voreinstellung ISO 8859-1 die richtige Wahl sein, sie deckt alle westeuropäischen Sprachen ab.
    • Zeilenumbruch nach: Hiermit legen Sie Zeilenumbrüche für die Textversion Ihrer Mailings fest. Als Standard ist ein Umbruch nach 72 Zeichen voreingestellt.
    • Format: Hier können Sie den Versand an Empfänger eines bestimmten Mail-Formats auswählen:
      • Text: Im Textformat enthält die E-Mail keinerlei Formatanweisungen wie fett oder Schriftgröße und auch keine klickbaren Links. Dafür sind alle E-Mail-Programme in der Lage, E-Mails im Textformat anzuzeigen.
      • HTML: Das HTML-Format erlaubt die bei Webseiten üblichen Formatierungen und auch die Integration von Bildern. Bei externen Dateien (Bildern etc.) enthält die E-Mail nur die Referenz, die Bilddatei selbst ist auf dem AGNITAS-Server gespeichert. Manche E-Mail-Programme und Mail-Provider mit Web-Browser-Schnittstelle wie GMX haben Problem mit der Anzeige solcher E-Mails.
      • Offline-HTML: Offline-HTML funktioniert genauso, allerdings sind Bilder und andere externe Dateien Teil der E-Mail (als Dateianhänge).
  • Ggf. Reiter SMS
    • Tragen Sie hier eine Absenderkennung ein.
  • Ggf. Reiter MMS
    • Betreff: Tragen Sie einen Text ein, der später beim Versand als Betreffzeile Ihrer MMS an den Empfänger erscheinen soll.
    • Absender-Adresse: Über dieses Feld bestimmen Sie, welche Adresse als Absender-Adresse standardmäßig im abschließenden MMS-Versand erscheint.
  • Ggf. Reiter Fax
    • Betreff: Tragen Sie einen Text ein, der später beim Versand als Betreffzeile Ihres Faxes an den Empfänger erscheinen soll.
    • Auflösung: Stellen Sie hier die Qualität der Auflösung (Normal oder Fein) Ihres Faxes ein.
  • Ggf. Reiter Print
    • Papiergröße: Wählen Sie hier das Format des Printprodukts (DIN A4, DIN A5 oder DIN A6).
    • Qualität: Stellen Sie hier die Qualität der Auflösung (Normal oder Fein) Ihres Printerprodukts ein.
  • Betreff: Wenn Sie ein neues Mailing auf Basis dieser Schablone anlegen, wird dieser Text als Vorschlag eingeblendet. Sie können ihn dann übernehmen, oder durch einen anderen Text ersetzen. Die Wirkung des Betreff sollte nicht unterschätzt werden. Er sollte gut gewählt werden, damit der Empfänger das Mailing öffnet. Beachten Sie, dass hier auch personalisierte Angaben mit den AGNITAS-Tags erlaubt sind.
  • Absender-Adresse: Wenn Sie ein neues Mailing auf Basis dieser Schablone anlegen, wird dieser Text als Vorschlag für die Absender-Adresse eingeblendet. Sie können ihn dann übernehmen, oder durch einen anderen Text ersetzen. Anzuraten ist eine nachvollziehbare E-Mail-Adresse wie z.B. newsdienst@ihreadresse.de. Allgemein erwecken Real-Name-Absender mehr Vertrauen beim Empfänger, also etwa sebastian.mueller@firma.de.
  • Zeichensatz: Die Auswahlliste zeigt die Zeichensätze an, die der E-Marketing Manager zur Verfügung stellt. Für die meisten Fälle dürfte die Voreinstellung ISO 8859-1 die richtige Wahl sein, sie deckt alle westeuropäischen Sprachen ab.
  • Zeilenumbruch nach: Diese Auswahlliste ist nur für Textmails interessant: Der E-Marketing Manager fügt nach der angegebenen Zeichenzahl automatisch einen Umbruch ein.
  • Format: Hier wählen Sie das Mail-Format der Schablone aus: Text-, HTML- oder Offline-HTML. Im Textformat enthält die E-Mail keinerlei Formatanweisungen wie fett oder Schriftgröße und auch keine klickbaren Links. Das HTML-Format erlaubt die bei Webseiten üblichen Formatierungen und auch die Integration von Bildern. Bei externen Dateien (Bildern etc.) enthält die E-Mail nur die Referenz, die Bilddatei selbst ist auf den Agnitas-Server gespeichert. Offline-HTML funktioniert genauso, allerdings sind Bilder und andere externe Dateien Teil der E-Mail (Dateianhänge).
  • Text-Version: In diesem Eingabefeld fügen Sie die Text-Version der Schablone ein. Weitere Details dazu lesen Sie im Handbuch. Auch wenn Sie in der vorhergehenden Auswahlliste das HTML oder Offline-HTML als Format gewählt haben, tragen Sie hier einen Text ein. Denn auch die Empfänger von HTML-Mails erhalten zur Sicherheit auch die Textversion.
  • HTML-Version: In diesem Eingabefeld fügen Sie die HTML-Version der Schablone ein. Weitere Details dazu lesen Sie im Handbuch.
  • Speichern": Speichert Ihre Einstellungen bzgl. der überarbeiteten Schablone.
  • Kopieren: Erstellt eine Kopie der jeweiligen Schablone und erleichtert die Erstellung von lediglich leicht modifizierten Schablonen.
  • Löschen: Löscht nach nochmaliger Rückfrage die Schablone.
