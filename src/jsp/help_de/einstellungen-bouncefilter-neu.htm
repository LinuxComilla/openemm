Online-Hilfe: Einstellungen - Bounce-Filter - Neuer Bounce-Filter
Über diesen Eingabedialog erstellen Sie einen neuen Bouncefilter. Füllen Sie die Eingabefelder aus und klicken Sie dann auf den Button Speichern. Über den Button Die Eingabefelder haben folgende Bedeutung:
  • Name: Der Name unter dem dieser Bounce-Filter gefunden werden soll.
  • Beschreibung: Die Beschreibung ist nur in der Übersichtsliste der Bounce-Filter sichtbar.
  • Eingehende E-Mails weiterleiten: Aktivieren Sie dieses Feld, wenn Sie eingehende E-Mails an eine bestimmte E-Mail-Adresse weiterleiten wollen.
  • Weiterleitungs-Adresse: Wenn Sie Eingehende E-Mails weiterleiten aktiviert haben, geben Sie hier die Zieladresse für die Weiterleitung an.
  • Autoresponder: Aktivieren Sie dieses Feld, wenn Sie eingehende E-Mails automatisch mit einer Standardemail beantworten wollen.
  • Absender-Adresse: Wenn Sie Autoresponder aktiviert haben, tragen Sie hier die Emailadresse ein, die als Absender für die automatische Antwort verwendet werden soll.
  • Betreff: Tragen Sie einen Text für die Betreffzeile der automatisch generierten Antwort ein.
  • Text-Version: Für automatisch generierte Antwortemails tragen Sie hier den Inhalt für die Text-Version ein.
  • HTML-Version: Für automatisch generierte Antwortemails tragen Sie hier den Inhalt für die HTML-Version ein.
