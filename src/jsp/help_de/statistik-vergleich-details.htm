Online-Hilfe: Statistik - Vergleich - Details

Dieser Eingabedialog präsentiert die Mailings, die Sie für einen Vergleich ausgewählt haben. Zu jedem Mailing sind einige Kennzahlen wie die Anzahl der Abmeldungen oder der Empfänger insgesamt aufgeführt. Eine detaillierte Statistik zu einem Mailing öffnen Sie mit einem Klick auf den Namen oder die Beschreibung. Mit dem Symbol Export () übertragen Sie die statistischen Daten als csv-Datei auf Ihre Festplatte. Eine solche csv-Datei können Sie mit einer Tabellenkalkulation wie Excel oder einem Datenbankprogramm öffnen. Über den Button Zurück gelangen Sie wieder zur Auswahl der Mailings, die Sie vergleichen wollen (wie mit Statistik - Vergleich).

  • Zielgruppe: Über diese Auswahlliste schränken Sie die Empfänger ein, die für die Statistik zählen. Anschließend klicken Sie auf den Button OK. Damit stellen Sie zum Beispiel fest, wie ein Mailing bei der männlichen Kundschaft zwischen 20 und 30 angekommen ist - sofern Sie diese genauen Daten in Ihrer Profildatenbank haben.
