Online-Hilfe: Aktionen - Neu Aktion
Über diesen Eingabedialog erzeugen Sie eine neue Aktion. Eine Aktion, bzw. die darin festgelegten Arbeitsschritte, führt das System aus, wenn der Empfänger eines Mailings auf einen messbaren Link in der E-Mail klickt, ein Formular absendet oder sowohl durch einen messbaren Link in einem E-Mail oder eine Absendung eines Formulars ausgelöst wird. Im oberen Bereich legen Sie Name und Beschreibung der Aktion fest. Im unteren Bereich erzeugen Sie neue Arbeitsschritte für die Aktion. Eine Aktion darf aus beliebig vielen Arbeitsschritten bestehen. Die allgemeinen Eingabefelder haben folgende Bedeutung:
  • Name: Der Name der Aktion erscheint in verschiedenen Auswahllisten des E-Marketing Managers, wenn Sie eine Aktion auswählen müssen. Sie sollten also einen kurzen, aber präzisen Namen festlegen.
  • Beschreibung: Die Beschreibung erscheint nur in der Übersicht der Aktionen. Es ist etwa sinnvoll, kurz die einzelnen Arbeitsschritte zusammenzufassen, aus denen die Aktion besteht.
  • Verwendung: Hier legen Sie fest, wofür Sie die Aktion einsetzen wollen. Mit der Voreinstellung "nur für Links" legen Sie fest, dass diese Aktion nur bei Link-bezogenen Themen in den Auswahllisten erscheint. So bleibt die Anzahl der Einträge in der Auswahlliste kleiner. Die Einstellung "nur für Formulare" bewirkt, dass die Aktion nur in Auswahllisten für Formulare sichtbar ist. Mit der Auswahl "Links und Formulare" ist die Aktion in allen Fällen sichtbar.
Ein Arbeitsschritt kann eine von mehreren Aktionen ausführen: etwa eine Profil-Modifikation, das An- oder Abmelden von einer Mailingliste, das Versenden einer E-Mail oder einer Double-OptIn-E-Mail oder das Laden von Profildaten. Sie stellen im Bereich Schritt hinzufügen die gewünschte Option über die Auswahlliste ein. Danach sichern Sie den ersten Arbeitsschritt mit dem Button Speichern. Ab dem zweiten Arbeitsschritt klicken Sie statt dessen auf den Button Hinzufügen. Jetzt zeigt der E-Marketing Manager je nach ausgewählter Aktion andere Details an.
  • Profil-Modifikation: Eine Profil-Modifikation verändert die Daten zu einem Empfänger in der Profildatenbank. Das ist vor allem mit selbst definierten Feldern sinnvoll. Wenn Sie mit einem Klick auf den Button Speichern eine neue Profil-Modifikation erzeugt haben, zeigt der E-Marketing Manager einen neuen Arbeitsschritt an. Über zwei Auswahllisten und ein Eingabefeld regeln Sie nun, welche Änderungen am Profil erfolgen sollen. Die Bedienelemente haben folgende Bedeutung:
    • Spalten-Name: In dieser Auswahlliste stellen Sie ein, welches Feld des Profils bearbeitet werden soll. Das System zeigt einige interne Felder und alle selbst definierten Felder an. Die internen Felder haben folgende Bedeutung:
      • FIRSTNAME: Der Vorname des Empfängers.
      • LASTNAME: Der Nachname des Empfängers.
      • TITLE: Der Titel des Empfängers (Dr. etc.).
      • GENDER: Das Geschlecht des Empfängers. Der E-Marketing Manager verwendet Zahlen zur Kennzeichnung des Geschlechts: Eine 0 steht für männlich, eine 1 steht für weiblich. Mit der Ziffer 2 sind alle Empfänger markiert, bei denen kein Geschlecht festgelegt wurde.
      • MAILTYPE: Das Mail-Format, das der Empfänger wünscht. Eine 0 steht für Text, eine 1 für HTML, eine 2 für Offline-HTML.
      • BOUNCELOAD: Zähler, der misst wie oft ein Soft Bounce aufgetreten ist. Ein Bounce ist eine E-Mail, die als unzustellbar an den AGNITAS-Server zurückgeschickt wurde.
      • TIMESTAMP: Das Datum der letzten Änderung am Empfängerprofil.
      • CREATION_DATE: Das Datum, an dem der Empfänger in die Datenbank eingetragen wurde.
      • DATASOURCE_ID: Kennziffer der Datenquelle, aus der die Empfängerdaten importiert wurden. Beim Einlesen von csv-Daten über die Importfunktion weist der E-Marketing automatisch eine ID zu (siehe Handbuch).
    • Operator: Es stehen drei Operatoren zur Verfügung. Das Plus-Symbol addiert zum Feldinhalt den im Eingabefeld angegebenen Zahlenwert. Das Minus-Symbol subtrahiert den Zahlenwert vom Feldinhalt. Beide Operatoren sind nur mit numerischen Feldern sinnvoll, nicht mit Text- oder Datumsfeldern. Der Gleichheitsoperator überträgt den Wert aus dem Eingabefeld in das Profil. Dabei dürfen Sie sowohl Zahlen als auch Texte oder ein Datum angeben.
    • Wert: In diesem Eingabefeld am Ende der Zeile tragen Sie den Wert ein, mit dem das Profil modifiziert werden soll.
  • Versand: Der Arbeitsschritt löst den Versand eines zuvor erstellten Mailings an denjenigen Empfänger aus, der den messbaren Link angeklickt und/oder ein Formular abgeschickt hat (je nach Zuordnung unter "Verwendung). Über zwei Auswahllisten stellen Sie die Details zum Versand ein:
    • Mailing: In dieser Auswahlliste stellen Sie das Mailing ein, das an den Empfänger versandt werden soll. Dieses Mailing müssen Sie vorher als Eventbasiertes Mailing über die Rubrik Mailings - Neues Mailing erzeugen. Bitte beachten Sie: Wenn Sie in der Auswahlliste Mailing nichts sehen, liegt es meistens daran, dass Sie das Eventbasierte Mailing nicht im Register "Versand" aktiviert haben
    • Verzögerung: : Es ist nicht immer sinnvoll, dass das Mailing sofort versandt wird, nachdem der Empfänger den messbaren Link angeklickt hat. Sie können Verzögerungszeiten zwischen 1 Stunde und 7 Tagen auswählen.
  • Anmelden: Der Arbeitschritt überträgt Daten in die Empfängerliste in einer Mailingliste.
    • Standardmäßig ist in dieser Aktion das Feld "Doublettenabgleich" aktiviert. Über das Drop-Down-Menü "Schlüsselspalte für Dubletten-Erkennung und Aktualisierungen" geben Sie die jeweilige Schlüsselspalte ein. Als Standard ist hier die E-Mail-Adresse als Schlüsselspalte zur Erkennung von Doubletten vorgesehen. Durch Auswahl innerhalb des DropDown-Menüs kann diese Standardeinstellung geändert werden.
    • Wenn ein Empfänger sich zunächst durch ein Double-Opt-In in die Mailingliste eintragen soll, dann aktivieren Sie bitte das Feld "Double-Opt-In verwenden".
  • Double-Opt-In Bestätigung Löst nach dem Klick auf einen messbaren Link oder dem Absenden eines Formulars eine Profiländerung in der Datenbank aus, die den Empfänger von einem ruhenden Newsletterempfänger auf einen durch Double-Opt-In-Bestätigung aktiv gewordenen Empfänger wandelt.
  • Profildaten laden: Dieser Arbeitsschritt löst nach einem Klick auf einen messbaren Link oder dem Absenden eines Formulars die Anzeige des betreffenden Profils aus. Diese Funktion hilft z. B. bei der Selbstumstellung des Empfängers von Text auf HTML-Mails.
  • Abmelden: Dieser Arbeitschritt Arbeitsschritt löst nach einem Klick auf einen messbaren Link oder dem Absenden eines Formulars die Abmeldung eines Empfängers von einer Mailingliste aus.
Mit dem abschließenden Klick auf "Speichern" speichern sie die jeweilige Aktion endgültig. Nach Klick auf Löschen erscheint eine Rückfrage, in der Sie diesen Vorgang nochmals bestätigen müssen bzw. diesen auch abbrechen können.
