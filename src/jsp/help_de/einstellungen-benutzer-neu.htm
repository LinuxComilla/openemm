Online-Hilfe: Einstellungen - Benutzer - Neuer Benutzer
In diesem Eingabedialog legen Sie einen neuen Benutzer an. Ein kurzes Wort zum Unterschied zwischen Mandant und Benutzer, wie er im E-Marketing Manager aufgefasst wird: Der Mandant hat umfassende Zugriffsrechte auf die AGNITAS-Software, ist sozusagen der Primärnutzer. Jeder Mandant hat eine eigene Profildatenbank für die Empfänger und eigene Einstellungen für Mailings etc. Er selbst kann sekundäre Benutzer anlegen und ihnen spezielle (aber auch umfassende) Rechte zuweisen. Wenn Sie mit Ihren Eingaben fertig sind, klicken Sie auf den Button Speichern. Die Eingabefelder und Auswahllisten haben folgende Bedeutung:
  • Voller Name: Der Name des Benutzers, zum Beispiel Vor- und Nachname oder ein Firmenname. Sonderzeichen und Leerzeichen sind dabei erlaubt.
  • Benutzer-Name: Der Benutzer-Name dient beim Login in den E-Marketing Managers zur Anmeldung. Er darf keine Leerzeichen oder Sonderzeichen enthalten, wohl aber Zahlen.
  • Passwort: In dieses Eingabefeld tragen Sie das Passwort ein, mit dem sich der Benutzer beim Login anmelden muss. Beim Eintippen zeigt der Browser statt der realen Zeichen zur Sicherheit nur Sternchen an. Achtung: Leerzeichen im Passwort sind nicht erlaubt.
  • Bestätigen: Um Tippfehler beim Passwort auszuschließen, wiederholen Sie in diesem Eingabefeld das Passwort. Beim Eintippen zeigt der Browser statt der realen Zeichen zur Sicherheit nur Sternchen an.
  • Sprache: Über diese Auswahlliste stellen Sie die Sprache der Benutzeroberfläche des E-Marketing Managers ein. Es stehen eine englische und eine deutsche Oberfläche zur Verfügung. Es ist kein Problem, wenn zwei Benutzer eines Mandanten unterschiedliche Spracheinstellungen benutzen, die Empfängerprofile und Einstellungen sind davon nicht betroffen.
  • Zeitzone: Über diese Auswahlliste stellen Sie sicher, dass alle Benutzer dieselbe Uhrzeit verwenden, auch wenn sie sich in verschiedenen Zeitzonen aufhalten.
  • Mandant: Der Benutzer ist einem Mandanten fest zugeordnet. Die Auswahlliste zeigt alle im E-Marketing Manager definierten Mandanten an. Wählen Sie den Mandanten aus, dem Sie den Benutzer zuordnen wollen.
