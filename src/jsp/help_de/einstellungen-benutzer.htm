Online-Hilfe: Einstellungen - Benutzer
Diese Übersicht zeigt Ihnen die vorhandenen Benutzer an. Ein kurzes Wort zum Unterschied zwischen Mandant und Benutzer, wie er im E-Marketing Manager aufgefasst wird: Der Mandant hat umfassende Zugriffsrechte auf die AGNITAS-Software, ist sozusagen der Primärnutzer. Jeder Mandant hat eine eigene Profildatenbank für die Empfänger und eigene Einstellungen für Mailings etc. Er selbst kann sekundäre Benutzer anlegen und ihnen spezielle (aber auch umfassende) Rechte zuweisen. Deshalb ist ein Benutzer immer einem bestimmten Mandanten zugeordnet, der in der Liste in der dritten Spalte aufgeführt ist. Um die Details zu einem Benutzer aufzurufen, klicken Sie entweder auf den Namen oder am Ende der Zeile auf das Symbol Bearbeiten ().
