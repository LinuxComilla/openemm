Online-Hilfe: Mailings - Versand
Das Register Versand zeigt den Status des Mailings und erlaubt Ihnen, die E-Mails zu prüfen, testweise zu verschicken, oder das Mailing komplett abzusenden. Die Links haben folgende Bedeutung:
  • Vorschau: Die Vorschau ruft einen neuen Eingabedialog auf, der Ihnen eine E-Mail so präsentiert, wie sie beim Empfänger der Mailings ankommt.
  • Test-Mails an Administratoren: Der Link verschickt die E-Mail an die Empfänger, die das Mailing abonniert haben und in der Profildatenbank als Administrator markiert sind. Ein Administrator kann zum Beispiel ein Vorgesetzter sein, der das Mailing vor dem Versand freigeben muss.
  • Test-Mails an Testverteiler: Der Link verschickt die E-Mail an die Empfänger, die das Mailing abonniert haben und in der Profildatenbank als Testempfänger markiert sind. Damit überprüfen Sie anhand einiger ausgewählter Empfänger, ob das Mailing beim Empfänger so aussieht, wie Sie es wollen. Als Testempfänger verwenden Sie dazu E-Mail-Adressen, die Sie selbst oder Kollegen empfangen.
  • Mailing abschicken: Über diesen Link starten Sie den Versandvorgang an sämtliche Empfänger, die die Mailing-Liste abonniert haben (und den Zielgruppen entsprechen, falls Sie diese Funktion verwendet haben).
