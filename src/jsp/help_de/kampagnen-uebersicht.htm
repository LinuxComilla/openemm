Online-Hilfe: Kampagnen - Übersicht

In der Übersicht sehen Sie alle Kampagnen, die Sie im OpenEMM definiert haben. Die neueste Kampagne finden Sie ganz oben in der Liste. Um Details zu einer Kampagne anzuzeigen, klicken Sie den Eintrag in den Spalten Kampagne oder Beschreibung an.

Alternativ klicken Sie das Symbol Bearbeiten () am Ende der jeweiligen Zeile an. Um eine Kampagne zu löschen, klicken Sie auf das Symbol Löschen (). Die einzelnen Spalten haben folgende Bedeutung:

  • Kampagne: Der Name des Kampagne, den Sie beim Anlegen vergeben haben. Wenn Sie die Kampagne öffnen, können Sie den Namen verändern.
  • Beschreibung: Die Beschreibung gibt Auskunft über den Ablauf bzw. den Zweck der Kampagne, beim Anlegen einer neuen Kampagne tragen Sie den Text selbst ein. Wenn Sie die Kampagne öffnen, können Sie die Beschreibung verändern.
