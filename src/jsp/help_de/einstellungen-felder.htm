Online-Hilfe: Einstellungen - Felder verwalten
Dieser Eingabedialog zeigt eine Liste aller Felder an, die Sie selbst der Profildatenbank hinzugefügt haben. Um die Details zu einem Feld anzuzeigen, klicken Sie den Namen an oder das Symbol Bearbeiten () am Ende des Eintrags. Sie entfernen ein Feld mit dem Symbol Löschen () am Ende des Eintrags. Der E-Marketing Manager prüft vor dem Löschen, ob das fragliche Feld in anderen Funktionen des Programms verwendet wird. Wenn Sie beispielsweise eine Zielgruppe definiert haben, die das Feld benutzt, so akzeptiert das System den Löschauftrag nicht. Sie müssen dann zunächst die Zielgruppe löschen. Die einzelnen Spalten haben folgende Bedeutung:
  • Feldname: Der Name des Feldes, wie er in Auswahllisten zum Beispiel bei der Definition einer Zielgruppe erscheint.
  • Feldname in DB: Dieser Name erscheint nur bei der Verwaltung des Feldes, sonst zeigt ihn der E-Marketing Manager nie an. Unter diesem Namen ist das Feld in der Profildatenbank bei AGNITAS gespeichert, deshalb sind nur Großbuchstaben erlaubt.
  • Typ: Ein Feld kann unterschiedliche Arten von Daten aufnehmen. Es stehen folgende Feldtypen zur Verfügung:
    • Date: Felder dieses Typs speichern ein Datum. Die Eingabe erfolgt immer nach dem Schema JJJJMMTT, also beispielsweise 19650905 für den 5. September 1965.
    • Number: Der Feldtyp erlaubt als Feldinhalt "nur" Zahlen.
    • Varchar2: Bei diesem Feldtyp dürfen Sie beliebige Zeichen im Feld speichern.
