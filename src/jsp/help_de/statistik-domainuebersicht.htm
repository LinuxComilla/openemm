Online-Hilfe: Statistik - Domain-Übersicht
Dieser Eingabedialog sortiert die Empfänger in Ihrer Profildatenbank nach der E-Mail-Adresse. Dabei wertet das System nur die Bestandteile hinter dem At-Zeichen (@) aus, die sogenannte Domain. Damit stellen Sie fest, welche E-Mail-Provider Ihre Kunden bevorzugen. Interessanter wird es bei geschäftlichen E-Mail-Adressen, denn hier ist meistens der Firmenname in der Domain enthalten. Um die Auswertung einzugrenzen wählen Sie Zielgruppe und Mailing-Liste aus, danach klicken Sie auf den Button OK. Mit dem Symbol Export () übertragen Sie die statistischen Daten als csv-Datei auf Ihre Festplatte. Eine solche csv-Datei können Sie mit einer Tabellenkalkulation wie Excel oder einem Datenbankprogramm öffnen.
  • Zielgruppe: Über diese Auswahlliste schränken Sie die Empfänger ein, die für die Domain-Statistik zählen. Anschließend klicken Sie auf den Button OK. Damit stellen Sie zum Beispiel fest, welche Domains männliche Empfänger zwischen 20 und 30 bevorzugen - sofern Sie diese genauen Daten in Ihrer Profildatenbank haben.
  • Mailing-Liste: Über diese Auswahlliste schränken Sie die Domain-Statistik auf eine einzelne Mailing-Liste ein. Anschließend klicken Sie auf den Button OK. Nur Empfänger, die diese Liste abonniert haben, wertet der OpenEMM dann aus.
