Online-Hilfe: Mailings - Grafik-Komponenten
Eine E-Mail kann auch Bilder und andere Grafiken enthalten. Diese Bilder übertragen Sie mit diesem Eingabedialog zunächst auf den Server. Danach stehen die Grafik-Komponenten für den Einbau in ein Mailing zur Verfügung. Die Eingabefelder und Buttons haben folgende Bedeutung:
  • Neue Komponente: In das Eingabefeld tragen Sie den Dateinamen der Grafik oder des Bildes auf Ihrer Festplatte ein. Der Button Durchsuchen ruft den Dateiauswahldialog von Windows auf, der ausgewählte Dateiname erscheint dann im Eingabefeld. Der OpenEMM unterstützt die Grafikformate GIF und JPG. Um die Übertragung zum Server zu starten, klicken Sie auf den Button Hinzufügen.
  • Übersichtliste: In der Liste sind sämtliche für dieses Mailing bereits übertragenen Grafiken oder Bilder aufgelistet. Links sind bei jedem Eintrag der Dateiname und der Typ der Datei (Grafikformat) angegeben. Rechts sehen Sie eine Vorschau der Grafik. Über den Button Löschen entfernen Sie eine Grafik-Komponente aus dem Mailing. Achtung: Das System löscht die Komponente ohne R&ückfrage.
