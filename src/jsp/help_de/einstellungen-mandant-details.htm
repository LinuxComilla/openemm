Online-Hilfe: Einstellungen - Mandant - Details
Über diesen Eingabedialog ändern Sie die Einstellungen zu einem Mandanten und erzeugen neue Benutzer. Der Dialog ist zweigeteilt: Oben sind die Eigenschaften des Mandanten aufgelistet, unten steht die Liste der Benutzer, die diesem Mandanten zugeordnet sind. Wenn Sie mit den Änderungen fertig sind, klicken Sie auf den Button Speichern.
  • Name: Der Name des Mandanten sollte kurz sein und keine Sonderzeichen enthalten.
  • Beschreibung: Die Beschreibung erläutert knapp den Zweck des Mandanten.
  • COMPANY_ID: Das interne Kennzeichen eines Mandanten ist die COMPANY_ID. Der E-Marketing Manager vergibt dieses Kennzeichen automatisch bei der Einrichtung, der Wert lässt sich entsprechend nicht verändern. Sie benötigen diese ID für die Gestaltung der Webseiten des An- und Abmelde-Interfaces.
Der untere Bereich Benutzer des Eingabedialogs listet die Benutzer auf, die dem Mandanten zugeordnet sind. Für jeden Benutzer gibt es eine Zeile. Sie rufen die Details zu einem Benutzer auf, wenn Sie den Namen anklicken oder das Symbol Bearbeiten () am Zeilenende. Über den Link Neuer Benutzer rufen Sie den entsprechenden Eingabedialog auf. Der neue Benutzer ist dann diesem Mandanten zugeordnet.
