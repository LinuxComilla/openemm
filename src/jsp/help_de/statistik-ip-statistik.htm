Online-Hilfe: Statistik - IP-Statistik

Hier stellen Sie fest, über welche IP-Adressen sich Anwender für eine Mailing-Liste angemeldet haben. Der OpenEMM zeigt die 20 häufigsten IP-Adressen an. Um die Auswertung einzugrenzen wählen Sie Zielgruppe und Mailing-Liste aus, danach klicken Sie auf den Button OK. Mit dem Symbol Export () übertragen Sie die statistischen Daten als csv-Datei auf Ihre Festplatte. Eine solche csv-Datei können Sie mit einer Tabellenkalkulation wie Excel oder einem Datenbankprogramm öffnen.

Die Spalte IP-Adresse zeigt die entsprechende IP-Adresse an. In der Spalte Empfänger listet das System auf, wie viele Kunden sich über diese Adresse angemeldet haben.

  • Zielgruppe: Über diese Auswahlliste schränken Sie die Zielgruppe ein, die für die IP-Statistik berücksichtigt werden soll. Anschließend klicken Sie auf den Button OK.
  • Mailing-Liste: Über diese Auswahlliste schränken Sie die IP-Statistik auf eine einzelne Mailing-Liste ein. Anschließend klicken Sie auf den Button OK. Nur Empfänger, die diese Liste abonniert haben, wertet der OpenEMM dann aus.
