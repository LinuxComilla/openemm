Online-Hilfe: Einstellungen - Anredeart - Neue Anredeart
Über diesen Eingabedialog erzeugen Sie eine neue, individuelle Anredeart. Füllen Sie die Eingabefelder aus und klicken Sie dann auf den Button Speichern. Dabei weist der E-Marketing Manager der neuen Anredeart automatisch eine ID, eine Kennzeichnung, zu. Über die ID und das AGNITAS-Tag [agnTITLE type=xx] binden Sie die Anredeart in ein Mailing ein. Statt xx tragen Sie die ID der gewünschten Anredeart ein. Über den Button Abbrechen gelangen Sie zur Übersicht der Anredearten. Die Eingabefelder haben folgende Bedeutung:
  • Beschreibung: Die Beschreibung ist nur in der Übersichtsliste der Anredearten sichtbar, die Empfänger der Mailings sehen sie nie.
  • Anrede (Gender=0, männlich): In das Eingabefeld tragen Sie die gewünschte Anrede für männliche Empfänger ein, zum Beispiel Lieber Herr. Den Nachnamen ergänzen Sie später bei der Definition eines Mailings mit einem AGNITAS-Tag.
  • Anrede (Gender=1, weiblich): In das Eingabefeld tragen Sie die gewünschte Anrede für weibliche Empfänger ein, zum Beispiel Liebe Frau. Den Nachnamen ergänzen Sie später bei der Definition eines Mailings mit einem AGNITAS-Tag.
  • Anrede (Gender=2, unbekannt): In das Eingabefeld tragen Sie die gewünschte Anrede für Empfänger unbekannten Geschlechts ein, zum Beispiel Lieber Interessent.
