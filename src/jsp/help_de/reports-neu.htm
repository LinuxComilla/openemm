Online-Hilfe: Reports
Über diesen Eingabedialog richten Sie einen neuen Report ein. Wenn Sie mit den Eingabe fertig sind, klicken Sie auf den Button Speichern. Sämtliche Eingaben können Sie später über die Detailansicht der Mailing-Listen auch wieder ändern.

Wenn Sie einen Report anlegen und diesen sich regelmäßig zusenden lassen möchten, dann können Sie dies wie folgt tun:

  • Neuen Report einrichten Klicken Sie im neuen Menüpunkt "REPORTS" auf "NEUEN REPORT". Dort geben dem Report einen Namen und eine kurze Beschreibung ein.

    Der Name eines einmal eingerichteten Reports erscheint später in der REPORT-Übersichtliste. Achten Sie deshalb darauf, aussagekräftige Texte zu verwenden. Das erleichtert Ihnen später die Arbeit, wenn Sie verschiedene Reports eingerichtet haben und diese ändern bzw. löschen möchten.

    Als nächstes geben Sie die E-Mail-Adresse an, an die der Report zukünftig versendet werden soll und bestimmen den Wochentag und die Uhrzeit, an dem dieser erstellt und Ihnen zugestellt werden wird.

  • Module für Reports festlegen Hier legen Sie fest, aus welchen Modulen Ihr Report bestehen soll.

    Der Report kann aus mehreren Modulen bestehen. Die Module unterscheiden sich in Ihrer Zusammensetzung grundsätzlich zunächst in der Art ihres Typs. Der Typ legt fest, ob das Modul die Statistik eines bestimmtes Mailings oder aber einen Vergleich mehrerer Mailings miteinander umfassen soll (Fenstermenü "Typ"). Nach der Auswahl im entsprechenden Fenstermenü öffnet der Klick auf "Hinzufügen" das "Report-Modul".

    Dort geben Sie im Fenstermenü an, ob Sie in diesem Modul einen Überblick über einzelne Mailingstatistiken für "das aktuelle", "das letzte" oder "das vorletzte" Mailing (Fenstermenü "nach Mailings") erhalten wollen oder ob Sie sich "Zwischenstände" Ihrer Mailings zukommen lassen möchten.

    Sie können unter Mailing-Statistik "nach Zeitraum" sich aber auch den Stand noch am Versandtag, nach einem, zwei oder mehreren Tagen zuschicken lassen.

    Als weitere Option für das jeweilige Modul können Sie die Statistik auf bestimmte Kampagnen und Zielgruppen begrenzen (durch Auswahl des entsprechenden Fenstermenüs).

    Mit Klick auf "Speichern" fixieren Sie das erste Modul Ihres Reports.

  • Weitere Module dem Report hinzufügen Wenn Sie Ihren Report um weitere Module erweitern möchten, dann können Sie dies, indem Sie den betreffenden Report durch Klick auf "Hinzufügen", Eingabe neuer Modul-Variablen und Klick aus "Speichern" ergänzen.

  • Weitere Reports anlegen Selbstverständlich können Sie auch mehrere Reports anlegen und sich diese sich selbst oder Ihren Kollegen - auch an unterschiedlichen Tagen und Zeitpunkten - zuschicken lassen.

    Hierzu gehen Sie wie oben unter "Neuen REPORT einrichten" beschrieben vor.

    Die Zusendung der einzelnen Reports erfolgt dann zukünftig solange nach dem gleichen Schema bis Sie den Report entsprechend ändern oder aber löschen. Ändern und Löschen der Reports können Sie unter "Reports-Übersicht". Eine Löschung der gesammelten Daten innerhalb der Statistik erfolgt dadurch selbstverständlich nicht.
