Online-Hilfe: Einstellungen - Inhalt-Quellen - Neue Inhalt-Quellen
Über diesen Eingabedialog erzeugen Sie eine neue Inhalt-Quelle. Dabei handelt es sich um die URLs von XML-Dateien, die der E-Marketing Manager automatisch in neue Mailings integrieren kann. Wenn Sie diese Funktion nutzen wollen, sprechen Sie mit uns. Wenn Sie mit Ihren Eingaben fertig sind, klicken Sie auf den Button Hinzufügen. Die einzelnen Eingabefelder haben folgende Bedeutung:
  • Inhalt-Quelle: Der Name der Inhalt-Quelle sollte kurz und prägnant sein. Er erscheint im E-Marketing Manager im Inhalt-Dialog für eines Mailings in den Auswahllisten, die alle verfügbaren Quellen anzeigen.
  • Beschreibung: Die Beschreibung kennzeichnet in einem knappen Satz den Sinn der Inhalt-Quelle. Der Text erscheint in der Übersicht der Inhalt-Quellen.
  • URL: Die URL ist die Web-Adresse, von der sich der E-Marketing Manager die zu importierende XML-Datei herunterladen kann. Dabei kann es sich zum Beispiel um die Adresse eines FTP-Servers oder eines Web-Servers handeln. Achtung: Der E-Marketing Manager prüft zunächst nicht, ob die Adresse gültig ist. Erst beim Anlegen eines neuen Mailings mit Inhalt-Quelle lädt das System die angegebene Datei.
