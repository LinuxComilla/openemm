Settings - Type of salutation Detail

Click on the Settings tab in the navigation bar, then on the Forms of salutation submenu. A list of defined forms of address is displayed. To edit an existent entry, click on the edit-button The entry fields have the following meaning:

  • ID: The OpenEMM assigns this ID automatically. The ID is used to select a form of address. Use the AGNITAS-TAG [agnTITLE type=xx] to include this address in the mailing. Instead of xx you put in the ID of the particular address. The ID can't be changed at a later point.
  • Description: Enter a meaningful name into the Description field.
  • Address: (Gender=0, male) insert the address for male recipients. Example: Dear Mr. The last name can be added later on within the definition of a new mailing.
  • Address: (Gender=1, female) insert the address for female recipients. Example: Dear Ms. The last name can be added later on within the definition of a new mailing.
  • Address: (Gender=2, unknown) insert the address for unknown recipients. Example: Dear Customer
