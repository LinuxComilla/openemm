Settings - Profilefields

Personal data for each mailing recipient are stored in a separate profile in the OpenEMM database. To display the details of a field, click on the name or on the edit-button at the end of the line. The delete-button will delete the field. OpenEMM checks if a feature of the program is still using the field, before deleting. You may not delete a field, what is still in use for example by a targetgroup. In this case, you first have to delete the targetgroup.
The singular fields have the following meaning:

  • Field name: The name of the field, which appears in the options menu, for example in target group definitions.
  • Fieldname in DB: This name only appears in the field management. This field specifies the name which will be saved in the profile database.
  • Type: A field can include different kinds of data. There are the following field types available:
    • Date: Fields of this type save a date. The entry has to be made with YYYYMMDD. For example 20080301for March 01 2008.
    • Numerical: This field type allows the entry of numbers.
    • Alphanumerical: There are now restrictions for your entry by choosing this field type.
