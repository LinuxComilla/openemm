Settings - Type of salutation

You choose whether to greet your customers more formally or whether you prefer a more relaxed style of address. The system distinguishes between male and female recipients as well as those whose sex is unknown.

The Overview lists all types of salutation, you already defined. By clicking on the edit-button you can see all the details of the type and can edit them. If you want to deleted one specific type of salutation, just click on the delete-button at the end of the line. With confirming the request, the type will be deleted from the system

  • ID: The ID is a label for one specific type of address. The ID is used to select a form of address and to include this address in the mailing by using the AGNITAS-TAG [agnTITLE type=xx]. In place of xx you put in the ID of the particular address.
  • Example (ID 1): [agnTITLE type=1] produces the following salutation for the recipient John Smith: Dear Mr Smith
