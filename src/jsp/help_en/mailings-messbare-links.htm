Mailings - trackable links

In order to evaluate the success of a mailing, trackable links may be inserted into the text. As soon as a mailing recipient clicks on the link, it calls up a URL linking to the OpenEMM server. The server registers a click and assigns it to a recipient. This works both for text mailings and for HTML mailings. However, the difference is in the notation. The example contains an integrated link. A trackable link measures whether or not a mailing recipient clicked on that link. You can then measure whether or not the recipient was interested in the mailing subject.

  • URL: this field shows the webaddress, in the mailing. It has to be a valid address to any web server. To check this link, click on the arrow-button in front of the entry. The browser opens a new window/tab with your defined URL.
  • Description: Here, you can insert a short text describing your link. The mailing recipient cannot see that description. In the statistical evaluation, the OpenEMM does, however, display the description. This simplifies evaluation where long URLs are involved.
  • Trackable:
    • Non-trackable: The link appears unchanged in the mailing. The recipient is able to click on the link, but OpenEMM can't analyse this action.
    • Text only: this option means that the link is no longer trackable in the html version, making it useless for statistics. This option should therefore only be used if a certain link must appear within the mailing, for instance the company's mail address
    • HTML only: The link will be only formatted trackable in HTML and Offline-HTML mailings. The recipients with text-mails see the original link like in the option "non-trackable"
    • Text and HTML: The link will be formatted trackable in all mailings. It usually makes sense to choose this option.
  • Action: This column shows, how OpenEMM reacts, when a user clicks on a trackable link. This action has to be defined already under the category actions.

By clicking on an URL, on a description or on the edit-button at the end of the line, OpenEMM will open a window with all details of the trackable link.
