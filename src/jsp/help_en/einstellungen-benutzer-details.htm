Settings - User Details

For each account you may create individual users, assigning them specific user rights. Each account must have at least one user who logs in when registering with the OpenEMM. These users are listed in Settings - Users. There you can edit the user details. Once you have made all your entries, click on the save button

  • Full name: it takes user's first and last names, or a company name
  • Username: a special name the user enters when logging into the OpenEMM. It should contain no spaces or special characters, but can contain numbers.
  • Password: Enter a Password in the relevant box. For security reasons, web browsers do not show characters but stars for that password. For this reason, the password must be confirmed by entering it again.
  • Language: there is a choice between the German, English or French software versions.
  • Time zone: if a user is in a different time zone, this may be entered here.
  • Client: assigns the user to a client account.
