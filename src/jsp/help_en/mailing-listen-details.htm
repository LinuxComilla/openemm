Mailing lists - Details

Mailing lists are the key element for sending out newsletters using the OpenEMM. A mailing list contains all prospects and subscribers for an individual mailing or a newsletter. The mailing list manages all declarations of acceptance (or permission information) which recipients have given when requesting E-Mails. For each newsletter you will create an individual mailing list. The fields have the following meaning.

  • Name: Each time you have to select a mailing list, the OpenEMM will display this text. Recipients, however, will not see this text.
  • Description: You should try to use meaningful descriptions which will make your work easier once you have to manage many different mailing lists. Especially if several employees use the OpenEMM, unambiguous mailing list names are a must.
