Statistic - Mailing statistic Details

The statistical overview gives you a list of all newsletters sent so far, in the Mailing statistics section. Simply click on one of the newsletters in the Mailing column to select it for evaluation. You are immediately shown both in graphic and in numerical form how many clicks there were to the links in your mailing. The same evaluation is accessed by selecting a mailing from the Mailings section and then clicking on the Statistics tab.

  • Targetgroup: By choosing the targetgroup you decide for which part of the recipients you want to get the statistic. You can even compare up to 5 target groups according to their statistic by adding several targetgroup with the add-button.
    Additional details are available on all E-Mails which were opened. Clicking on the link Opened E-Mails will cause OpenEMM to display data according to domains taken from recipients' E-Mail addresses and rank the 20 top domains by number of recipients.
  • URL:The first passage lists all trackable links in the mailing. In the first column OpenEMM shows the web address, in the second column you can see the number of registered clicks. By using the arrow in front of the web address you can visit the target page in a new browser window.
  • Total clicking subscriber: this graph shows the proportion between the recipients and the clicks.
  • Opened Mails:This number is only valid for HTML-Mails that have the AGNITAS tag [agnONEPIXEL] included in the HTML text module.
  • Opt-Outs:Tells you how many recipients used the opt-out link in the mailing.
  • Bounces: Tells you how many recipients used the opt-out link in the mailing. Bounces: This value shows how many E-Mails had a wrong recipient address or other problems receiving the mailing.
  • Recipient:The number of recipients to whom the E-Mails got send.

You may clean the statistic, this will filter all admin- and testmails. The export-button allows you to export the statistic in a csv-file.
