Settings - New Bounce-Filter

Click on "Settings" on the left part of OpenEMM. In the submenu select "Bounce-Filters". On the right side the existing bounce-filters are shown. Click on "New Bounce-Filter" above the list.

  • Name: The field "Name" is shown in the list of bounce-filters.
  • Description: Enter some descriptive text in this field
  • Forwarding incoming mailings: Activate the field "Forward incoming E-Mails" and enter an E-Mail address in "Address" if you want to forward incoming E-Mails to that address.
  • Forwarding address: The address for forwarding should be an E-Mail address which is read regularly to allow quick reaction on important E-Mails.
  • Autoresponder: Activate the field "Autoresponder" and edit the values in "Sender-Address", "Subject", "Text version" and "HTML version" to define a generated response E-Mail.
  • Subject: Enter a text for the subject line of the autogenerated mailing.
  • Text version: Insert here the content for autoresponder mailings in text format
  • HTML version: Insert here the content for autoresponder mailings in HTML format.
