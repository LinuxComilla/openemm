Mailings - Overview

All existing Mailings are sorted by there delivery date (latest on top). You can edit the particular E-Mail by clicking on the name, or on the edit-button. The delete-button deletes the mailing from the system.

  • Mailing: The name of the Mailing you choose at its creation.
  • Description: You can use the Description field to add further information, helping you to distinguish your mailing according to internal procedures
  • Mailing list: A Mailing list has to be created already. You will start by determining which Mailing list your mailing should be based on. In other words, you will select the recipients who feature that particular mailing list in their profile.
  • Send date: The date when the Mailing got delivered. If this field is empty the mailing has not been send yet.
