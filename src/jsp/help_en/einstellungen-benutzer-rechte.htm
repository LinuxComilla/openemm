Settings - Assigning user rights

You may now assign individual user rights to an existing user. Single-click on the user name of the user you want to assign individual rights to. The system will now display an entry dialog where you can change individual user settings (name, password etc.).
Click on the User rights tab: The system will display a list of possible user rights. All are individually or groupwise selectable. Just click on the corresponding check box to assign that particular right. Once you have entered your selections, click on the Save button. User rights may be modified or withdrawn at a later date.
