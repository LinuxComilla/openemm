Settings - New User

For each account you may create individual users, assigning them specific user rights. Each account must have at least one user who logs in when registering with the OpenEMM. You call up the entry dialog for creating a new user via the navigation bar, in Settings, sub-menu Users, then open the New user tab in the contents area.
Fill in the required data in the input boxes. Full name takes user's first and last names, or a company name. User name, on the other hand, is a special name the user enters when logging into the OpenEMM. It should contain no spaces or special characters, but can contain numbers.
Enter a Password in the relevant box. For security reasons, web browsers do not show characters but stars for that password. For this reason, the password must be confirmed by entering it again.
Several drop-down lists are used to enter the following information:

  • Language: there is a choice between the German, English or French software versions.
  • Time zone: if a user is in a different time zone, this may be entered here.

    • Once you have made all your entries, click on the Create button. Once you have created the new user, the OpenEMM displays an overview of all users existing in the system.
